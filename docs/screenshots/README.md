# CRM Screenshots Directory

This directory is used to store screenshot images of the running Mini CRM application. These images are linked in the main [README.md](../../README.md) file to show your project's UI on GitHub.

## 📸 How to populate this folder:

1.  Run the project locally (`npm run dev` and `php artisan serve`).
2.  Open [http://localhost:5173](http://localhost:5173) in your browser.
3.  Take screenshots of the following pages and save them in this folder with the **exact** file names:
    *   **Login Page:** Save as [login.png](file:///c:/xampp/htdocs/miniCRM/docs/screenshots/login.png)
    *   **Dashboard Page:** Save as [dashboard.png](file:///c:/xampp/htdocs/miniCRM/docs/screenshots/dashboard.png)
    *   **Customers List:** Save as [customers.png](file:///c:/xampp/htdocs/miniCRM/docs/screenshots/customers.png)
    *   **Deals Kanban Board:** Save as [deals.png](file:///c:/xampp/htdocs/miniCRM/docs/screenshots/deals.png)
4.  Once saved, commit and push these files to your GitHub repository:
    ```bash
    git add docs/screenshots/
    git commit -m "docs: added application UI screenshots"
    git push origin main
    ```
